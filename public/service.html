Service

できること

HTML5

FLOCSSの命名規則に従い記述し保守性の高いコーディングを心がけています。ピクセルパーフェクトを元にデザインを忠実に再現しています。

CSS3

柔軟なレスポンシブ対応を行います。デザインカンプにもよりますが基本的に320px~1290pxで表示崩れが発生しないように心がけています。
Sassを使用してコーディングをしております。コードの可読性を上げるために入れ子は基本的に使用しておりません。

JavaScript (jQuery)

主にjQueryを使用しdrawerやトップページへのスクロールやアコーディオンなどWebサイトでよく使用されるパーツの実装が可能です。
プロジェクトによりますがEJS,Pugを使用してのコーディングも可能です。

WordPress

オリジナルテーマの実装が可能です。
クライアント様が管理しやすいサイト作成や、情報更新に柔軟に対応したカスタム設定が可能です。

Git

ローカル環境で作業し、変更履歴を追跡し異なるバージョンのコードを管理しています。

Github

Githubを使用してリモートでソースコードを共有、管理しております。
